Flatten(IEnumerable<Rung>)

Returns a flat list of NeutralText representing all base and nested AOI logic in the collection of Rung objects.

public static IEnumerable<NeutralText> Flatten(this IEnumerable<Rung> rungs)

Parameters

rungs IEnumerable<Rung>

Returns

IEnumerable<NeutralText>

A IEnumerable<T> containing all the NeutralText, including nested instruction text, found in the rung collection.

Remarks

This extension was specifically created to assist in getting a flat list of logic, including nested AOI logic, for specialized querying purposes, such as finding tag references within nested logic. This method will replace the instruction logic parameters with the neutral text operands of the instruction signature, so to get the effective flattened list of executing NeutralText code.
