ScopeName(Tag)

Returns the TagName and the tag's program name prepended with the format Program:{ProgramName}.{TagName} if the tag is a program tag. If the tag is a controller tag, then returns the tag's TagName. This forms a globally unique tag identifier for the given tag object.

public static TagName ScopeName(this Tag tag)

Parameters

tag Tag

Returns

TagName

If this tag is a controller tag, then TagName. If this tag is a program tag, then ContainerName(Tag) and TagName in the format Program:{ProgramName}.{TagName}.

Remarks

This value is determined based on the Scope of the tag. This name provides a consistent global unique identified for a given tag object.

Tags(LogixContent)

Returns all Tag objects in the entire L5X, including controller, program, AOI, and module tags.

public static TagDictionary Tags(this LogixContent content)

Parameters

content LogixContent

The LogixContent representing the L5X file to query.

Returns

TagDictionary

A IEnumerable<T> containing all Tag objects in the L5X.

Remarks

Tags are scattered throughout the L5X file. The API of this library sort of reflects that with all the different places to look for tag objects (controller, programs, modules). This extension is here to support a single interface through which to retrieve a flat list of all tags in the L5X regardless of container and element name. This method supports controller, program, AOI, and module config, input, and outputs tags.
