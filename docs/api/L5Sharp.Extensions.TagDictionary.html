Remarks

This class is optimized to quickly retrieve tag objects by name using nested dictionaries. This is important simply for the fact that a single L5X file could reasonably contain millions of tag objects when considering that even atomic members can have up to 64 member tags.

This class is quick to create and quick to retrieve, and should probably be used in place of .NET ToLookup() of ToDictionary() as they will not be as performant, especially with larger collections.

Remarks

This performs a quick lookup of the tag object using a unique tag name. Given that program scoped tag names are not unique, tagName must be prefixed with the program name specifier in the format Program:{ProgramName}, where ProgramName is the name of the program the tag is contained in.
