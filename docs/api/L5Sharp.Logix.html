Module(string, string, string?)

Creates a new Module with the provided name and catalog number.

public static Module Module(string name, string catalogNumber, string? description = null)

Parameters

name string

The name of the module

catalogNumber string

The catalog number to lookup a catalog entry for.

description string

The option description of the module.

Returns

Module

A new Module object initialized with data return by the catalog service.

Remarks

This factory method uses the ModuleCatalog service to lookup info for the specified catalog number. If RSLogix is not installed on the current environment, this will throw an exception.

Exceptions

InvalidOperationException

The module catalog service could not load the installed catalog database file -or- catalog number does not exist in the catalog database.

ArgumentException

catalogNumber is null or empty.
