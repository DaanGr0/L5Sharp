A thin wrapper around the textual representation of logic notation called neutral text. This could represent a single instruction signature, a rung of logic (combination of instructions), or a line of structured text. The purpose of this class is to provide a way to parse the text into strongly typed objects that are easier to work with.

Remarks

Neutral text can represent a single instruction or a full rung (collection of instructions). Each instruction contains sets of tag names and values known as arguments or operands. This class provides functions for extracting the textual information into strongly type classes that are easier to work with.
