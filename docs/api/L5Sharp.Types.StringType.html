Remarks

A logix string type has predefined members LEN and DATA, which contain the current string length and set of ASCII characters, respectively. This class is inherited by STRING, which is Rockwell's built in base string type. You may also create instance of this class providing a string value and type name.

StringType has special cases in terms of it's L5X structure. Rockwell treats strings sort of like a value type, giving it it's own DataFormat. However, when serialized as a member of a complex structure, the data looks more like a generic structure type.
