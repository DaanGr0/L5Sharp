Remarks

This object is a simple entity type wrapper that groups the vendor id and name. Vendor's are defined by Rockwell and assigned unique Id and name. This information is obtained from the L5X or from the ModuleCatalog service by lookup of a given catalog number. Use Rockwell as it is the most common vendor for compatible devices.
