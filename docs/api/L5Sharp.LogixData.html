ScanMode

The ScanMode specifying how LogixData should perform reflection scanning for types to pre-register for deserialization.

public static ScanMode ScanMode { get; set; }

Property Value

ScanMode

Remarks

This class will perform reflection based scanning for types matching the all the following criteria: Public non-abstract classes that derive from LogixElement or LogixType and have public constructors accepting a XElement object. Types match this criteria will be cached and constructed at runtime.

Scan(Assembly)

Scans the provided assembly using reflection for public non-abstract types inheriting StructureType that have the required deserialization constructor and registers the type so that it may be instantiated during deserialization of a L5X.

public static void Scan(Assembly assembly)

Parameters

assembly Assembly

The assembly to scan.

Remarks

This is to assist with easily registering types within a specific assembly. Note that if ScanMode is set to All, all assemblies in the app domain will automatically be scanned. Therefore, only use this method if ScanMode is set to Internal or None.

Exceptions

InvalidOperationException

A type is already registered.
